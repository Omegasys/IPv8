# Quality of Service (QoS) Specification

## Overview

IPv8 includes QoS mechanisms for traffic prioritization.

## Core Components

### Flow Label

* Identifies packet flows
* Enables consistent routing behavior

### Priority Levels

* Low
* Normal
* High
* Critical

### Traffic Types

* Real-time (voice/video)
* Bulk transfer
* Control traffic

## Features

* Flow-aware routing
* Congestion handling
* Latency-sensitive prioritization

## Goals

* Improve performance for critical applications
* Reduce jitter and packet loss
