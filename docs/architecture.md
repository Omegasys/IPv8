Architecture

IPv8 is composed of several layers:

1. Core Layer
Addressing
Packet format
Header processing
2. Crypto Layer
Hashing (SHA-256)
Encryption (AES or similar)
Key exchange
IPsec integration
3. Network Layer
Routing logic
Path selection
Direction-aware routing
4. Transport Adapters
TLS support
WebSocket support
5. Compatibility Layer
IPv4/IPv6 translation
Dual-stack handling
6. Tools & CLI
Debugging
Simulation
